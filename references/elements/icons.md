## 图标使用原则

1. **有意义的手绘元素**：图标与内容强相关，必须放在指定位置
2. **手绘风格一致**：所有图标用手绘线条，不用 emoji
3. **颜色克制**：主要使用蓝色，最多 1-2 个其他颜色点缀
4. **描述清晰**：prompt 中要描述图标的具体形态（如"两个重叠的矩形"）

## 图标 + 文字组合示例

```
左侧手绘圆形编号"1"（蓝色粗边框），右侧内容：
- 手绘复制图标（两个重叠的矩形）
- 文字："点击豆包返回内容中的"复制"按钮，或手动全选代码复制"
- 下方用浅蓝色手绘线条框展示简化的代码块示意（3-4行等宽字体模拟代码）
```
