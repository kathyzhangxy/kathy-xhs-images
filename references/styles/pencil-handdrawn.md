# 铅笔手绘风（Pencil Hand-drawn）

> 所属 Skill：kathy-xhs-images
> 版本：v1.0
> 创建日期：2026-04-05

## 风格概述

Notion 风格的铅笔手绘线条，清爽干净的知识卡片视觉。核心特征：白色背景 + 蓝色铅笔线条 + 局部彩色点缀。

## 风格参数

| 参数 | 值 | 说明 |
|------|-----|------|
| **风格标识** | `pencil-handdrawn` | 用于 prompt 变量引用 |
| **背景** | 纯白（#ffffff） | 不做大面积实色填充 |
| **主色** | 蓝色（#2196f3） | 标题、边框、主要线条 |
| **点缀色1** | 橙色（#ff9800） | 关键词高亮、小星星、强调元素 |
| **点缀色2** | 紫色（#9c27b0） | 次要点缀、特殊色块 |
| **强调色** | 红色（#f44336） | 警告、关键步骤、问题标注 |
| **功能色** | 绿色（#4caf50） | 推荐标签、对勾图标 |
| **辅助色** | 黄色（#ffeb3b） | 荧光笔效果、高亮色块 |
| **代码块色** | 浅灰色（#f5f5f5） | 代码/示例背景 |

## 线条特征

- **线条风格**：手绘感铅笔线条，非机械直线
- **线条粗细**：标题框粗线条，内容框中等线条，连接元素细线条
- **边框样式**：手绘圆角矩形框，不规则感
- **连接元素**：手绘虚线箭头/曲线

##  Typography 特征

- **标题**：超大号粗体（48px），蓝色，醒目
- **正文**：中等字号（18px），蓝色或深灰色
- **副标题**：较小字号，蓝色
- **关键词高亮**：橙色手绘圆圈圈出
- **代码/示例**：等宽字体，浅灰色背景框

## 点缀元素风格

| 类型 | 风格化描述 |
|------|-----------|
| 小星星 | 橙色手绘五角星，2-3个 |
| 小彩带 | 橙色或紫色手绘彩带，斜向飘落 |
| 小圆圈 | 蓝色或橙色手绘圆圈，大小不一 |
| 小点 | 蓝色手绘小圆点，随机分布 |
| 小线条 | 蓝色或橙色手绘短线条，随机角度 |

## 色块使用规则

- **不做大面积实色填充**：背景保持纯白，内容区域用手绘线条框组织
- **局部色块**：问题描述区（黄色荧光笔）、底部引导语（浅紫色淡淡涂抹）
- **色块风格**：不规则边缘，像荧光笔/水彩笔涂抹效果，透明度低

## Prompt 风格约束模板

```
纯白背景。铅笔手绘线条风（notion风格），蓝色铅笔线条。
标题蓝色，内容用蓝色手绘线条框组织，不要大面积实色填充背景。
用手绘小图标代替emoji。
颜色克制，主要使用蓝色，点缀橙色和紫色。
不要画页码、系列名、网格线、徽章。
页面内容从上到下均匀分布，不要集中在上半部分。
```

## 适用场景

- 操作教程、跟着做系列
- 问题解决思路、调试指南
- 产品功能介绍、效果展示
