# 极简暖色风（Minimal Warm）

> 所属 Skill：kathy-xhs-images
> 版本：v1.2
> 创建日期：2026-04-06
> 更新日期：2026-04-06

## 风格概述

极简暖色调设计，纯色暖白背景 + 橙色圆形编号 + 橙色/红色/绿色功能标签。核心特征：干净排版、大标题、色彩分层、信息清晰。

## 色彩系统精确规范

| 色彩元素 | 精确色值 | 使用场景 |
|---------|---------|---------|
| **背景** | `#f5f0e8` | 暖白色纯色背景，无网格无纹理 |
| **主文字（标题）** | `#333333` | 大标题、技能名等一级文字 |
| **次级文字（正文）** | `#555555` | 功能描述、列表项等二级文字 |
| **橙色强调** | `#ed8936` | 圆形编号、标语、关键数据、封面标题第2行 |
| **红色标签** | `#e74c3c` | "它能做什么"竖线+文字 |
| **绿色标签** | `#27ae60` | "核心能力"竖线+文字 |
| **辅助灰色** | `#999999` | 右上角标签、底部统计标签 |
| **分隔线** | `#d0c8bc` | 暖灰色细线（非标准灰#e0e0e0） |
| **冠军卡片背景** | `#fef0db` | 冠军页底部卡片浅橙色背景 |
| **封面按钮** | `#333333` | 黑色圆角胶囊背景 |

## 线条特征

- **线条风格**：干净直线，无手绘感
- **线条粗细**：细线分隔（1px），用于内容区块分隔
- **边框样式**：无框或极简细线框，冠军页底部卡片用圆角细线框（圆角~10px）
- **分隔线**：暖灰色细线（`#d0c8bc`），用于列表项之间、标题区和内容区之间

### 文字颜色分层

- **一级文字（标题/技能名）**：`#333333`（深灰/近黑）
- **二级文字（描述/列表项）**：`#555555`（中灰）
- **三级文字（辅助/标签）**：`#999999`（浅灰）

### 行高规范

- **功能描述正文**：行高 1.8（非1.6）
- **核心能力列表**：行高 1.6
- **其他文字**：行高 1.5

## 橙色圆形编号规范

- **形状**：完美圆形
- **颜色**：橙色（`#ed8936`）实心填充
- **阴影**：极轻微阴影（opacity 5-8%, blur 10px），营造轻微悬浮感
- **尺寸**：
  - 内页（P1-P7）：直径 40-45px
  - 总览页（P8）：直径 30px（更小更紧凑）
- **内容**：
  - 内页：白色粗体数字带井号（`#7`、`#6`...`#1`），约24px
  - 总览页：白色粗体纯数字（`1`、`2`...`7`），约16px
- **位置**：
  - 在主标题和细线分隔符**下方**，距分隔线约25px
  - 左对齐，距左边缘约20px

## 红绿功能标签规范

**红色标签（"它能做什么"）：**
- 红色竖线 `|`（约2px宽，16px高，`#e74c3c`）+ 红色文字"它能做什么"
- 整体作为功能描述区的小标题
- **不配合任何图标**，纯文字+竖线
- 位于标语下方约20px，核心能力上方

**绿色标签（"核心能力"）：**
- 绿色竖线 `|`（约2px宽，16px高，`#27ae60`）+ 绿色文字"核心能力"
- 整体作为核心能力列表的小标题
- **不配合任何图标**，纯文字+竖线
- 位于功能描述下方约15px，列表上方

## 列表项圆点规范

- **形状**：标准小圆点（•）
- **颜色**：橙色（`#ed8936`）
- **尺寸**：直径6-8px
- **位置**：每个核心能力列表项左侧
- **间距**：圆点与文字之间约8px间距
- **数量**：固定4项（所有内页都是4个核心能力）

## Emoji 图标使用规范

| 场景 | Emoji | 颜色 | 间距规范 | 说明 |
|------|-------|------|---------|------|
| 底部统计-星标 | ⭐ | 黄色 | 与数字间距4-6px | 格式：`⭐ 1` |
| 冠军页-奖杯 | 🏆 | 金色 | 与文字间距4px | 冠军页底部卡片行首 |
| 冠军页-皇冠 | 👑 | 金色 | 与文字"冠军"间距2px | 冠军页右上角 |
| 封面-机器人 | 🤖 | 灰色/彩色 | 与文字间距4px | 封面底部左侧 |
| 封面按钮圆点 | • | 橙色 | 与文字间距4px | "必装清单"左侧小圆点 |

**注意：** 红绿功能标签**不使用** emoji，仅用竖线+文字。

## 底部统计区域规范

**普通页（P1-P7，#7到#1）：**
- 3组数据横向排列，间距均匀
- 格式：`大数字` + `小字标签`（如"115K" + "下载量"）
- 第3组是 `⭐ emoji` + `数字`（如"⭐ 1"）
- 数字黑色粗体（30px，`#333333`），文字浅灰色（14px，`#999999`）
- 距画布底边约30-40px

**冠军页（P7，#1）特殊处理：**
- 浅橙色背景卡片（`#fef0db`，圆角~10px，细线边框）
- 内容整合为一行：`🏆 244K 下载量 · 2.2K 安装 · 17 ⭐`
- 全部橙色文字（`#ed8936`，16-18px）
- 卡片下方补充语："社区公认的第一必装技能"（橙色，16px）

## 技能名称规范

- **字体**：等宽字体（monospace）
- **大小写**：全小写
- **连接符**：使用连字符（-），如 `skill-vetter`、`agent-browser`、`self-improving-agent`
- **颜色**：`#333333`（深灰）
- **字号**：30px

## 色块使用规则

- **纯色背景**：整个页面使用暖白色（`#f5f0e8`），不要网格、不要纹理
- **冠军页底部卡片**：浅橙色背景（`#fef0db`），圆角~10px，细线边框
- **封面底部按钮**：黑色圆角胶囊（`#333333`），白色文字，左侧橙色小圆点
- **不做大面积色块填充**：内容区域保持干净
- **橙色点缀**：仅用于圆形编号、标语、关键数据、底部统计数字

## 元素精确位置（从上到下）

1. **顶部标题区**：大标题左对齐 + 右上角灰色标签右对齐（**与大标题顶部在同一水平线**。主标题过长时缩小主标题字号至36-38px，确保右上角标签有空间同行显示。底部横跨细线分隔）
2. **编号圆圈**：细线下方约25px，左对齐，距左边缘约20px
3. **技能名**：圆圈下方约10px，左对齐
4. **标语**：技能名下方约5px，左对齐
5. **红色标签**：标语下方约20px，左对齐
6. **功能描述**：红色标签下方约8px
7. **绿色标签**：功能描述下方约15px
8. **核心能力列表**：绿色标签下方约8px，4项，每项行高约24px
9. **底部统计**：列表区下方约25px，3组横向等距排列

## Prompt 风格约束模板（完整版）

```
色彩系统精确规范：
- 背景：暖白色 #f5f0e8，纯色，无网格无纹理
- 主文字：深灰 #333333（标题、技能名），次级文字 #555555（描述、列表项）
- 橙色强调：#ed8936（圆形编号、标语、关键数据、封面标题第2行）
- 红色标签：#e74c3c（仅"它能做什么"竖线+文字）
- 绿色标签：#27ae60（仅"核心能力"竖线+文字）
- 辅助灰色：#999999（右上角标签、底部统计标签）
- 分隔线：暖灰色 #d0c8bc，1px细线
- 冠军卡片背景：浅橙色 #fef0db，圆角 ~10px

橙色圆形编号：#ed8936实心圆，白色"#N"文字（带井号），直径40-45px，极轻微阴影（opacity 5-8%, blur 10px），位于细线分隔符下方约25px，左对齐。
红色"它能做什么"标签：红色竖线| + 红色文字，不配合任何图标。
绿色"核心能力"标签：绿色竖线| + 绿色文字，不配合任何图标。
核心能力列表：固定4项，橙色小圆点•（#ed8936，直径6-8px），与文字间距8px。
底部统计数据：3组横向排列，数字黑色粗体30px，文字浅灰色14px，⭐emoji与数字间距4-6px。
技能名称：等宽字体（monospace），全小写，连字符连接（如skill-vetter），30px。
细线分隔：暖灰色 #d0c8bc，1px，用于标题区和内容区之间、列表项之间。
页面内容从上到下均匀分布，顶部标题区约15%，编号+技能名区约25%，功能描述区约20%，核心能力区约25%，底部统计区约15%。
不使用emoji代替红绿功能标签。

**顶部标题区对齐关键规则（v1.3 更新）：**
- 右上角标签与主标题**必须在底部基线对齐**（bottom baseline alignment）
- 由于右上角字号（14-16px）小于主标题（36-44px），右上角的顶部会自然低于主标题顶部，这是正常且美观的效果
- 当主标题文字较长时，**主动缩小主标题字号**，为右上角标签腾出空间：
  - ≤8 英文字符 / ≤6 中文字符：主标题 44px，标签 16px（正常布局）
  - 9-12 英文字符 / 7-8 中文字符：主标题 40px，标签 15px（适度缩小）
  - 13-18 英文字符 / 9-12 中文字符：主标题 36px，标签 14px（明显缩小）
  - >18 英文字符 / >12 中文字符：主标题 32px，标签 13px（显著缩小）
- Prompt 中必须使用精确术语：**"与主标题底部基线对齐"**，避免使用模糊的"同一水平线"（ImageGen 可能误解为垂直居中）
- **所有文字内容用引号包裹**，样式描述放在括号内（如 `"Summarize"（样式：大号粗体，深灰色，字号44px，左对齐）`），避免 ImageGen 将样式值渲染为文字
- 长标题需在 prompt 中注明缩小原因（如 `**标题较长(13字符)，从标准44px缩小至40px**`）
```

- 技能/资源介绍
