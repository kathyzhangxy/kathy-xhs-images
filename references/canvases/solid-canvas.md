# 纯色画布（Solid Canvas）

> 所属 Skill：kathy-xhs-images
> 版本：v1.0
> 创建日期：2026-04-06

## 画布概述

小红书标准竖版图片，3:4 宽高比，纯色背景，适合极简排版和信息清晰展示。

## 画布参数

| 参数 | 值 | 说明 |
|------|-----|------|
| **画布标识** | `solid-canvas` | 用于 prompt 变量引用 |
| **宽高比** | 3:4 | 小红书标准竖版 |
| **推荐分辨率** | 1080 x 1440 px | ImageGen 输出尺寸 |
| **方向** | 竖版（Portrait） | 从上到下阅读 |
| **背景类型** | 纯色 | 无网格、无纹理 |

## 安全区域

| 区域 | 边距 | 说明 |
|------|------|------|
| **顶部安全区** | 50px | 标题区域 |
| **左右安全区** | 40px | 内容不贴边 |
| **底部安全区** | 60px | 底部数据区域 |

## 纯色布局特征

- **垂直流式布局**：内容从上到下排列
- **极简分区**：用细线或留白分隔内容区块
- **信息密度**：中，适合清晰展示
- **阅读顺序**：从上到下

## Prompt 画布约束模板

```
竖版 3:4 小红书图片，1080x1440 分辨率。
纯色背景，不要网格、不要纹理。
内容从上到下排列，顶部留 50px 安全区，左右各留 40px 边距，底部留 60px 安全区。
```

## 未来画布扩展方向

| 画布类型 | 特征 | 适用场景 |
|----------|------|---------|
| `solid-canvas` | 纯色竖版 | 极简排版、排行榜、清单 |

| `split-screen` | 左右分屏 | 强烈对比、Before/After |
