# 网格画布（Grid Canvas）

> 所属 Skill：kathy-xhs-images
> 版本：v1.0
> 创建日期：2026-04-05

## 画布概述

小红书标准竖版图片，3:4 宽高比，适合多模块信息排列的网格布局。

## 画布参数

| 参数 | 值 | 说明 |
|------|-----|------|
| **画布标识** | `grid-canvas` | 用于 prompt 变量引用 |
| **宽高比** | 3:4 | 小红书标准竖版 |
| **推荐分辨率** | 1080 x 1440 px | ImageGen 输出尺寸 |
| **方向** | 竖版（Portrait） | 从上到下阅读 |

## 安全区域

| 区域 | 边距 | 说明 |
|------|------|------|
| **顶部安全区** | 60px | 标题区域，不被裁切 |
| **左右安全区** | 50px | 内容不贴边 |
| **底部安全区** | 80px | 底部提示区域 |

## 网格布局特征

- **垂直流式布局**：内容从上到下排列，适合多模块页面
- **模块化分区**：每个内容块独立容器，清晰边界
- **信息密度**：中高，适合干货知识、教程类内容
- **阅读顺序**：从上到下，从左到右

## 布局模式适配

当前画布支持以下布局模式：

| 布局 | 适配度 | 说明 |
|------|--------|------|
| 布局 A：浏览器预览 + 功能清单 | ✓✓ | 左侧预览 + 右侧清单，经典网格 |
| 布局 B：手机双屏对比 | ✓✓ | 左右并排手机屏幕 |
| 布局 C：操作步骤时间线 | ✓✓ | 垂直排列步骤卡片 |
| 布局 D：多方案卡片 | ✓✓ | 垂直排列方案卡片 |
| 布局 E：问题调试页 | ✓✓ | 顶部问题场景 + 中部方法卡片 |

## Prompt 画布约束模板

```
竖版 3:4 小红书图片，1080x1440 分辨率。
内容采用网格布局，从上到下分区排列。
顶部留 60px 安全区（标题），左右各留 50px 边距，底部留 80px 安全区。
```

## 未来画布扩展方向

| 画布类型 | 特征 | 适用场景 |
