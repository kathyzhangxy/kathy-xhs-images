# 生图检查清单

> 从 feedback-log、bad cases、postmortem 中提取的必查项
> 更新日期：2026-04-04

---

## 生图前检查（Step 1-3）

### 方向确认
- [ ] 已判断页面类型：Cover / Inner
- [ ] 已输出「生图思路」：布局框架 + 内容模块 + 视觉元素
- [ ] 生图思路已获得主管确认
- [ ] 已参考对应的 good case（Cover → good/cover/，Inner → good/inner/）

### Prompt 撰写
- [ ] 标题颜色明确指定为蓝色（"蓝色大号粗体标题"）
- [ ] 不使用 emoji，用手绘小图标代替
- [ ] 不使用实色填充，用手绘线条框
- [ ] 颜色克制：主色蓝色 + 最多1个点缀色
- [ ] 内容描述具体、丰富，确保页面不空
- [ ] 纯白背景，不描述网格/田字格
- [ ] 无需顶部留白（本模版不使用徽章）
- [ ] 不画页码、系列名、徽章

---

## 生图后检查（Step 4）

### 基本检查
- [ ] 图片尺寸正确（ImageGen 输出 1024x1536）
- [ ] 已立即移动到 temporary/P{N}-{页面名}/ 文件夹
- [ ] 已删除 vibe_images/ 中的源文件
- [ ] 文件命名正确：P{N}-{页面名}-V{版本}.png

### 内容检查
- [ ] 标题是蓝色，不是黑色
- [ ] 没有 emoji
- [ ] 没有实色填充的卡片/背景
- [ ] 颜色没有花花绿绿（主蓝 + 最多1点缀）
- [ ] 页面内容丰满，没有大面积空白
- [ ] 文字/元素数量准确（如数字与 C2 一致）
- [ ] 没有网格线（由 pipeline 添加）
- [ ] 没有页码/系列名/徽章

### Pipeline 检查
- [ ] 使用了 `--no-badges` 参数
- [ ] Pipeline 成功执行（无报错）
- [ ] 输出了网格背景 + 去白底合成效果

## 常见问题速查

| 问题 | 原因 | 解决 |
|------|------|------|
| 标题是黑色 | prompt 没指定颜色 | 明确写"蓝色大号粗体" |
| 页面太空 | 内容描述不够具体 | 增加模块、图标、细节描述 |
| 有实色背景 | prompt 允许了填充 | 明确"不要实色填充" |
| 有 emoji | prompt 没禁止 | 明确"用手绘小图标代替emoji" |
| 颜色太多 | 没限制点缀色 | 明确"最多1个点缀色" |
| 有网格线 | prompt 描述了网格 | 删除网格描述，由 pipeline 添加 |
| 顶部大片空白 | prompt 描述了留白 | 本模版无需留白，删除相关描述 |
| 内容偏浅/看不清 | 去白底阈值问题 | 调高 `--threshold 240` |
