# P0 - 大字报封面 Prompt V4

## 页面类型
Cover（大字报封面）- 淡紫色版本

## Prompt

```
竖版 3:4 小红书图片，1080x1440 分辨率。

淡紫色纯色背景（#E8D5F5），干净无纹理。

超大黑色粗体标题，居中排列，分两行：
第一行：这些指令让我（黑色粗体，约 56px）
第二行：秒变（黑色粗体，约 56px）豆包大神（黑色粗体，约 80px，比前面文字大很多，"豆包大神"四个字用黄色记号笔涂抹效果 #FFD93D 高亮）

标题垂直居中，占满画面中央区域，上下留白充分。

装饰元素：
- 左上角：一对淡紫色手绘双引号 " "（比背景色稍深一点，#C9B1DB）
- 右下角：手绘小图标组合（小星星✨ + 小爱心♥ + 小闪电⚡，淡紫色手绘风格，小尺寸，分散点缀）
- 底部右侧：一条短横线（淡紫色）

黄色高亮色块风格：不规则边缘，像黄色记号笔/荧光笔涂抹效果，半透明，不要完全遮盖文字。

极简设计，除了标题和装饰外，不要有任何其他元素。不要副标题、不要作者名、不要日期、不要系列名、不要页码、不要机器人图标。

所有文字黑色，不要彩色文字（除了黄色高亮色块）。
```

## 设计说明

- **风格**：极简大字报
- **背景**：淡紫色 #E8D5F5
- **标题**：黑色粗体，"豆包大神"80px 超大号 + 黄色记号笔高亮
- **装饰**：左上角引号 + 右下角手绘小图标（星星+爱心+闪电）+ 底部短横线
